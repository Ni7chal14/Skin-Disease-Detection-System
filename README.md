# Skin-Disease-Detection-System
Skin Disease Detection System using Deep Learning (ResNet50).  
This project classifies skin lesions into 7 categories using the HAM10000 dataset and is deployed as a Streamlit web application.

🧠 Skin Disease Detection System
This project is a Deep Learning-based web application that detects skin diseases from images using a trained ResNet50 model.


🚀 Features
Classifies 7 types of skin lesions
Uses HAM10000 dataset
Built using TensorFlow and Streamlit
Real-time image prediction

🛠️ Tech Stack
Python
TensorFlow / Keras
OpenCV
Streamlit

📊 Model Performance
Accuracy: ~72%
Balanced precision and recall
Handles class imbalance

⚠️ Note
Model works best on dermatoscopic images similar to the training dataset.

▶️ Run Locally
pip install -r requirements.txt
streamlit run app.py
